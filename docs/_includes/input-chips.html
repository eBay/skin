Input Chips in Disabled State

When disabled, input chips becomes "frozen" and users are unable to interact with the module - search for, add, and remove chips.

To turn on this state, add aria-disabled="true" to the outer element of the module. Additonally, you'll need to add the disabled attribute to all the buttons inside as well as the input element for combobox.

Input Chips in Error State

When a user enters text that is considered to be invalid, the input chips module can enter an error state. Developers can validate the text entry and trigger invalid/valid states as needed.

To turn on the error state, add a class of input-chips--error-state to the outer element of the module. To turn off the error state, remove the class.
