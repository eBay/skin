The Skeletons are simplified versions of layouts used on initial page load to indicate that information has not been fully loaded. Skeletons should only be used for loads that take 500ms or longer

The skeletons can be hidden using the html semantic attribute hidden before the content is ready.

NOTE: JavaScript is required to toggle visbility of the skeletons and content container.
