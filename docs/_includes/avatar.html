An avatar is used to show a user's state. The three states available are signed-in with image, signed-in witout image, and signed-out

When a user is signed in without an image, there can be a fixed color chosen to make the avatar bubble unque. Also each signed out avatar bubble can only have 1 letter at a time

The avatar should also label the state in aria-label, either as signed in with the username or signed out

Various colors

Avatars can have different colors. Colors can only be used with signed in - no profile pic. These colors should be consistent with a given username. The following colors are available.
avatar--teal (default)
avatar--light-teal
avatar--green
avatar--lime
avatar--yellow
avatar--orange
avatar--magenta
