Interactive Calendar

To make a calendar interactive, replace the span elements with button. Buttons are disabled directly instead of with a class and clipped text can be moved into aria roles, but all other attributes remain the same between static and interactive calendars. Keyboard focus management should be handled with JavaScript and a roving tab index.

{% include calendar-month.html interactive="true" index="1" %}
{% highlight html %}
... ...
August 2019
SuMoTuWeThFrSa
{% endhighlight %}
