Use the icon-btn class (for buttons) or the icon-link class (for links) and any of the available icons for a borderless, actionable icon style.

{% highlight html %} {% endhighlight %}

Badged Icon Button

An icon button can be badged using the badge module.

The button or anchor element requires an additional icon-btn--badged or icon-link--badged modifier, respectively.

{% highlight html %} {% endhighlight %}

Cart Nav Icon Button

DS Evo includes a new cart icon button with a grey background.

It requires an additional couple classes on the button - icon-btn--nav icon-btn--cart - as well as the icon-cart class on the svg.

{% highlight html %} {% endhighlight %}

Save Nav Icon Button

There is also a Save icon button.

It uses the icon-btn, icon-btn--badge, icon-btn--nav, and icon-btn--save classes on the button as well as the icon-save class on the svg.
