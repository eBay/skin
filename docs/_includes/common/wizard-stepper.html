Wizard steppers potray progress through a sequence of steps or may also be used for navigation.

Wizard steppers are best used

  1. Fixed sequence of data entry
  2. Guiding untrained users through a stepper process
  3. A series of independent branced data input
  4. Accomplishing a user goal that relies on sub-task completion

States of Wizard Stepper

The default wizard stepper has horizontally aligned list of steps.

  • Confirmation
  • Confirmation Bold
  • InProgress
  • Upcoming
  • 1 Default
  • Insufficient
  • Error

Default Wizard Stepper

The default wizard stepper has horizontally aligned list of steps.

Note that we have wizard-stepper__transition--* for each of the step list items (li.wizard-stepper__item). This will let the outgoing transition wizard step to match its color with the next wizard step state.

Vertical Wizard Stepper

The vertical wizard stepper has vertically aligned list of steps.

Just add wizard-stepper--vertical class to the .wizard-stepper container tag.

Fluid Wizard Stepper

Stepper width will increase as the screen size stretches with a maximum total individual step reaching 220px.

Just add wizard-stepper__items--fluid class to the ul.wizard-stepper__items tag.
