@ebay/skin/details

A details element is an interactive element used to show and hide content.

It uses the details and summary disclosure elements to show/hide content.

Everything inside details and below summary is considered the content.

Since IE/Edge do not support details and summary elements. The details-element-polyfill is used to solve this problem.

Details Summary with custom icon

The default details shown above uses an SVG background image, and therefore has a fixed colour. Foreground SVG, while slightly more verbose, honours the CSS cascade, thus allowing any possible colour for the icon. In the example, a custom style is added.

Required: set the hidden attribute on details__icon. This is because the class has a background-image icon by detault, and it needs to be hidden so only the foreground SVG is shown.
