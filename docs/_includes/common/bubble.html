@ebay/skin/bubble

The bubble module provides the base styles for a "speech bubble". A bubble like this is typically used in any type of flyout that "points" to another element or area of the page.

The bubble requires a mask element that partially masks the pointer element, and a cell element sets up some basic layout.

Default Bubble

A bubble's pointer is always set to the top and center by default.

Bubble Pointers

The pointer can be positioned using any one of the following modifiers:

  • bubble__pointer--top-left
  • bubble__pointer--top
  • bubble__pointer--top-right
  • bubble__pointer--bottom-right
  • bubble__pointer--bottom
  • bubble__pointer--bottom-left
  • bubble__pointer--left
  • bubble__pointer--right

Although we show all possible pointer positions in the demo below, the bubble would typically only ever have zero, one or two pointers at the same time.
