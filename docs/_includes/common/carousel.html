@ebay/skin/carousel

Carousels group together a list of similar items into a viewport of fixed height and width.

The viewport may contain a fixed, discrete subset of the list, or a continuous "as many as can fit" subset. A discrete carousel is typically created by providing the desired number of items-per-slide ahead of time for JavaScript. In either case, the number of visible items constitutes a "slide". Each carousel has a live region to convey slide updates to assistive technology.

NOTE: No JavaScript behaviour has been implemented in the carousel examples, we demonstrate the CSS aspect only. For detailed behaviour requirements, please visit the eBay MIND Pattern for Carousel.

Slideshow Carousel

A carousel with one item in the viewport is known as a slideshow carousel

Slideshow carousels can be discrete only. Apply the modifier carousel--slides to create a discrete carousel (this applies to discrete filmstrip carousels too).

The list items in a slideshow are typically set to 100% width.

NOTE: On touch screen devices the paddles will always be shown by default.
