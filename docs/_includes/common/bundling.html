Bundling

Skin supports Webpack and Lasso.js.

Webpack

Add @ebay/skin and @ebay/browserslist-config as dependencies to package.json. The Skin webpack modules will import CSS so make sure you have css-loader and style-loader on your dev dependencies.

{% highlight json %} { "dependencies": { "@ebay/browserslist-config": "^1", "@ebay/skin": "^{{ page.version | slice: 0, 2 }}", }, "devDependencies": { "css-loader": "^3", "style-loader": "^1" } } {% endhighlight %}

Now in your webpack configuration do:

{% highlight js %} module.exports = { module: { rules: [ { test: /\.css$/, use: [ 'style-loader', 'css-loader', ], }, ], }, }; {% endhighlight %}

Usage

{% highlight js %} import "@ebay/skin/button"; {% endhighlight %}

Lasso.js

Add the @ebay/skin module as a browser.json dependency to include the entire set of Skin modules.

{% highlight json %} { "dependencies": [ "@ebay/skin" ] } {% endhighlight %}

Or, alternatively, by specifying modules on a per-need basis.

{% highlight json %} { "dependencies": [ "@ebay/skin/less", "@ebay/skin/global", "@ebay/skin/button", "@ebay/skin/icon" ] } {% endhighlight %}
