Bundling

Skin is optimised for use with Lasso.js - an advanced JavaScript module bundler, asset pipeline and optimizer - but other bundlers are supported.

For Lasso.js, add the @ebay/skin module as a browser.json dependency to include the entire set of Skin modules.

{% highlight json %} { "dependencies": [ "@ebay/skin" ] } {% endhighlight %}

Or, alternatively, by specifying modules on a per-need basis.

{% highlight json %} { "dependencies": [ "@ebay/skin/less", "@ebay/skin/global", "@ebay/skin/button", "@ebay/skin/icon/foreground" ] } {% endhighlight %}
