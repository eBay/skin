@ebay/skin/switch

A switch behaves a bit like a checkbox - it can be on or off (i.e checked or unchecked). The key difference is that a switch is not a true form control. It should execute JavaScript on the client when toggled rather than submitting form data to the server.

A switch requires the switch role and aria-checked attribute for assistive technology.

Whereas checkboxes are often used to allow multi-selection from a group of choices, switches are more often used in isolation or as a series of unrelated options.

REMEMBER: every switch requires a visible label! Please see the field module for details on labelling controls.
