Toast Message

A toast must remain in a hidden state for all users and devices until called upon.

Show Toast
  • Use toast--show along with the toast container element
  • Remove HTML hidden attribute from the toast class container
Hide Toast
  • Use toast--hide along with the toast container element
  • Remove toast--show from the container element
  • Add HTML hidden attribute to the toast class container after a delay of 300 ms.

Mobile Layout

Use toast__mobile on smaller screens so that the toast message container will be edge to edge.
