@ebay/skin/flyout

The flyout module forms the basis of any generic overlay that "points" to another element or area of the page. It is also used to form the overlay for the flyout notice and tooltip modules.

The DOM order of the flyout in the page is very important! To avoid keyboard and screen reader accessibility issues, the flyout should immediately follow the element that it points to.

Flyouts are absolutely positioned by default. The flyout will be positioned relatively to the nearest ancestor with position:relative.

Default Flyout

The default flyout has it's pointer at the top center.

Flyout Pointers

The pointer can be positioned using any one of the following modifiers:

  • flyout__pointer--top-left
  • flyout__pointer--top
  • flyout__pointer--top-right
  • flyout__pointer--bottom-right
  • flyout__pointer--bottom
  • flyout__pointer--bottom-left
  • flyout__pointer--left
  • flyout__pointer--right

Although we show all possible pointer positions in the demo below, the flyout would typically only ever show zero, one or two pointers at the same time.
