Flags

Skin has the ability to switch between previous, current and upcoming design systems.

DS4 is currently the default design system. To opt into DS6, Lasso.js requires the skin-ds6 conditional dependency flag.

{% highlight JavaScript %} myLasso.lassoPage({ flags: ['skin-ds6'] }); {% endhighlight %}

Or, alternatively, using the Marko Taglib.

{% highlight html %} ... {% endhighlight %}

IMPORTANT: All modules have identical markup, whichever design system is chosen; however, not all LESS variables and mixins are supported in both systems, therefore be sure to reference the appropriate documentation.
