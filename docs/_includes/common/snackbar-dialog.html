A snackbar is a non-modal dialog spawned by the main web page or application.

Toggling the hidden attribute will hide and show the snackbar.

Transient Snackbar

The default snackbar displays a short message and dissapears after a minimum of 6 seconds.

Action Snackbar

A snackbar can contain a single call to action, but must remain on screen until explicitly dismissed (in order to give all users adequate time to find and reach the action).

A snackbar is non-modal and therefore should not capture or trap keyboard focus when opened. Keyboard users require a conveniant way to access the action. This is achieved via the accesskey attribute on each button.
