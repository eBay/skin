The expressive loader can replace our traditional spinner to feel faster during periods of longer loading. To portray responsive and quicker loads, the expressive loader's design uses visual momentum to indicate progression across refreshing, page loading and user actions It consists of an animated progress bar in eBay brand colors and an optional set of messages that are cycled into view. It also supports a reduced motion mode.

The expressive loader uses Javascript to cycle through messages and dynamically add new progress bar pieces with random widths. The following code examples represent snapshots of the expressive loader at any given time.

The expressive loader progress bar starts out with 4 lines in eBay brand colors. Their widths are random numbers between 10% and 40%.

Base Expressive Loader - Animating

Lines start out at 0 width and scale up to 10% - 40% the width of their container. When a line is finished scaling in, another line is added to the line group. Using Javascript, add animated lines:

  1. Create a new element with classes expressive-line-animation__line, expressive_line_animation__line--animated, and expressive-line_animation__line--${color} where ${color} is the next logical brand color.
  2. Randomly assign it a width between 10% and 40%.
  3. Listen for the element's animationend, then add another line and removes the oldest line.
  4. Prepend the line to the line container.

Expressive Loader with Messages

If there are multiple messages to display, cycle them in and out of view using Javascript. To cycle a message into view:

  1. Put the message content in the element with class expressive_loader__message--next.
  2. Add the expressive-loader__message-in class to animate that message into view.
  3. After the message animates in, duplicate its content into the element with expressive_loader__message--current.
  4. After a fixed amounto of time, add the expressive_loader__message--out class to the current message to animate it out of view. This can happen concurrently with the next message animating in.

Base Expressive Loader with Reduced Motion

When the user prefers reduced motion, the lines still animate. Instead of growing from 0% to a random size, they stay a fixed size and scroll slowly across the screen. No Javascript is needed to achieve this effect.
