Icon

The icon module allows direct access to the Skin iconography via the <svg> tag (i.e. foreground icons) or <span> tag (i.e. background icons).

  • Core Icons
  • Null Icons
  • Editing Icons
  • Selling Icons
  • Misc Icons

IMPORTANT: Skin removes all mouse and touch events on foreground icons due to a bug in IE. To add events to these icons you should wrap them in another element and attach your events there.
