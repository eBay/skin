@ebay/skin/listbox

The Skin listbox is intended for use as a custom view (or facade) for the native HTML select element. JavaScript will be required to create a 2-way binding between the two.

Note: The first item should be selected if the user does not specify a selection. This matches the behavior of a native select box where the first item will be selected if no selection has been specified.

Note: Because the listbox must grow to hold it's contents, we have chosen to have the listbox always remain fluid. It is up to the developer to contain the width, if necessary.
